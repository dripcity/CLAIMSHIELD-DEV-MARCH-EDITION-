# ClaimShield DV - Product Summary

ClaimShield DV is a SaaS platform that generates professional diminished value (DV) vehicle appraisal reports. The platform guides users through creating legally-defensible DV appraisals using the comparable sales method, leveraging AI to extract data from documents, automatically search for comparable vehicles, perform precise calculations using legally-recognized methodologies, and generate comprehensive 15-25 page PDF reports with state-specific legal citations.

## Core Functionality

- **8-Step Appraisal Wizard**: Guided interface for creating DV appraisals
- **AI Document Extraction**: Gemini-powered analysis of repair estimates, insurance docs, and vehicle documents
- **Automated Comparable Search**: Apify-based web scraping for vehicle comparables
- **Valuation Calculations**: Median-based comparable sales method with exact adjustment constants
- **PDF Report Generation**: Professional 15-25 page appraisal reports with state-specific legal citations
- **Document Templates**: Demand letters, bad faith calculators, expert affidavits
- **Payment Processing**: Stripe integration for report purchases and subscriptions
- **Email Delivery**: SendGrid integration for report delivery

## Key Users

- **Individuals**: Vehicle owners seeking DV claims
- **Appraisers**: Professional appraisers with USPAP certification needs
- **Attorneys**: Legal professionals managing DV cases
- **Body Shops**: Repair facilities offering DV services

## Technical Differentiators

- Exact calculation constants ($0.12/mile, 80% MSRP, 7% annual depreciation)
- Median-based valuation (not mean) for legally-defensible results
- State-specific legal citations (Georgia, North Carolina, generic)
- 5-level severity classification with NAAA grade mapping
- Private file storage with signed URLs
