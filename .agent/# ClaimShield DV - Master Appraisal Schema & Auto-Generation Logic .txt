# ClaimShield DV - Master Appraisal Schema & Auto-Generation Logic

## Document Purpose
This schema defines the complete data model and auto-generation rules for producing professional diminished value appraisal reports. The system is divided into **User Input Fields** (variable data) and **Auto-Generated Fields** (computed commentary, analysis, and calculations).

---

## SECTION 2: AUTO-GENERATED FIELDS (Computed Analysis)

### 2.1 Vehicle Valuation Calculations
```typescript
// AUTO-CALCULATED based on comparable vehicles
ValuationResults {
  // Pre-Accident Fair Market Value
  pre_accident_fmv: {
    raw_comp_values: number[] (adjusted values from each pre-accident comp)
    calculation_method: enum ("median" | "mean" | "weighted_average")
    calculated_value: number (median of adjusted comp values)
    confidence_range: {
      low: number (10th percentile)
      high: number (90th percentile)
    }
  }
  
  // Post-Repair Actual Cash Value
  post_repair_acv: {
    raw_comp_values: number[] (adjusted values from post-accident comps)
    calculation_method: enum ("median" | "mean" | "weighted_average")
    calculated_value: number (median of adjusted comp values)
    confidence_range: {
      low: number
      high: number
    }
  }
  
  // DIMINISHED VALUE (The Money Shot)
  diminished_value: {
    calculated_amount: number (pre_accident_fmv - post_repair_acv)
    percentage_of_pre_value: number ((DV / pre_FMV) * 100)
    percentage_of_repair_cost: number ((DV / repair_cost) * 100)
    
    // Cross-check with repair-cost method
    repair_cost_method_estimate: number (repair_cost * severity_coefficient)
    variance_analysis: string (auto-generate if methods differ >20%)
  }
}
```

### 2.2 Damage Severity Classification (AUTO-DETERMINED)
```typescript
// SYSTEM AUTO-CLASSIFIES based on input data
SeverityAnalysis {
  severity_level: enum (
    1 | // Minor: Cosmetic only, <10hrs labor, no structural
    2 | // Moderate: Bolt-on panels, 10-20hrs, no frame work
    3 | // Medium: Multiple panels, 20-35hrs, possible minor structural
    4 | // Major: Structural damage, 35-60hrs, frame pulling
    5   // Severe: Extensive structural, >60hrs, multiple frame components
  )
  
  // AUTO-GENERATED LOGIC for severity_level:
  // IF frame_pulling_required = true OR frame_labor_hours > 0 → severity ≥ 4
  // IF structural_damage = true AND total_labor_hours > 35 → severity = 4
  // IF total_labor_hours > 60 OR airbag_deployment = true + structural → severity = 5
  // IF total_labor_hours 20-35 AND no frame work → severity = 3
  // IF total_labor_hours 10-20 AND cosmetic only → severity = 2
  // IF total_labor_hours < 10 → severity = 1
  
  severity_justification: string (auto-generate based on logic above)
  
  // NAAA Post-Repair Grade (AUTO-DOWNGRADE from pre-accident)
  post_repair_naaa_grade: enum (
    "excellent" |    // Only if severity ≤ 1 AND pre-grade was excellent
    "good" |         // If severity ≤ 2 AND no structural
    "average" |      // If severity = 3
    "below_average" | // If severity = 4 (ANY structural work)
    "rough"          // If severity = 5
  )
  
  naaa_downgrade_reason: string (auto-generate explanation)
}
```

### 2.3 Comparable Vehicle Adjustments (AUTO-CALCULATED)
```typescript
// For EACH comparable vehicle, system auto-calculates adjustments
CompAdjustments {
  base_listing_price: number
  
  adjustments: {
    // Mileage Adjustment
    mileage_adjustment: number (
      // Industry standard: ~$0.10-0.15 per mile difference
      // Formula: (subject_mileage - comp_mileage) * $0.12
      // Positive if comp has MORE miles (increases comp value to match subject)
      // Negative if comp has LESS miles (decreases comp value)
    )
    
    // Equipment Adjustments
    equipment_adjustments: EquipmentAdjustment[] {
      item: string
      adjustment_factor: number (80% of factory MSRP)
      adjustment_value: number
      direction: enum ("add" | "subtract")
    }
    total_equipment_adjustment: number (sum of above)
    
    // Trim Level Adjustment (if not exact match)
    trim_adjustment: number (
      // Based on MSRP difference between trims
      // Only if trim is not exact match
    )
    
    // Year Adjustment (if not exact match)
    year_adjustment: number (
      // ~5-10% depreciation per year for newer vehicles
      // Only needed if comp year differs from subject
    )
    
    // Condition Adjustment (standardize all comps to "Good")
    condition_adjustment: number (
      // Apply NAAA condition grade adjustments
      // Excellent → Good: -5%
      // Average → Good: +5%
      // Below Average → Good: +10%
    )
    
    total_adjustments: number (sum of all adjustments)
    final_adjusted_value: number (base_price + total_adjustments)
  }
}
```

### 3.2 Auto-Generation Triggers
```typescript
// WHEN user clicks "Generate Report", system executes:

1. Validate all required fields are complete
2. IF use_auto_search = true:
   - Execute web scraping for pre-accident comps
   - Execute web scraping for post-accident comps
   - Apply adjustment calculations to each comp
3. Calculate pre_accident_fmv (median of adjusted comps)
4. Calculate post_repair_acv (median of adjusted comps)
5. Calculate diminished_value = pre_FMV - post_ACV
6. Auto-classify severity_level based on repair data
7. Auto-downgrade NAAA grade based on severity
8. Generate all narrative text fields using templates + variable substitution
9. Insert state-specific legal citations based on owner.address.state
10. Compile all sections into professional PDF
11. Attach uploaded documents as appendices
12. Email PDF to user + save to dashboard
```

## SECTION 4: CODING AGENT INSTRUCTIONS

### 4.1 Database Schema (PostgreSQL via Neon)
```sql
-- Use Drizzle ORM for type-safe database access

CREATE TABLE appraisals (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL REFERENCES users(id),
  
  -- Metadata
  claim_number TEXT,
  appraisal_date DATE NOT NULL DEFAULT CURRENT_DATE,
  accident_date DATE NOT NULL,
  purpose TEXT NOT NULL,
  status TEXT NOT NULL DEFAULT 'draft', -- draft | complete | sent
  
  -- Owner Info (JSONB for flexibility)
  owner_info JSONB NOT NULL,
  
  -- Insurance Info
  insurance_info JSONB,
  
  -- Subject Vehicle (JSONB for complex nested structure)
  subject_vehicle JSONB NOT NULL,
  
  -- Accident Details
  accident_details JSONB NOT NULL,
  
  -- Valuation Results (auto-calculated, stored for caching)
  valuation_results JSONB,
  
  -- Severity Analysis (auto-generated)
  severity_analysis JSONB,
  
  -- Appraiser Info (if applicable)
  appraiser_info JSONB,
  
  -- File Attachments (store GCS URLs)
  repair_estimate_url TEXT,
  damage_photos JSONB, -- array of GCS URLs
  repair_photos JSONB,
  
  -- Generated Report
  report_pdf_url TEXT,
  
  -- Timestamps
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE comparable_vehicles (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  appraisal_id UUID NOT NULL REFERENCES appraisals(id) ON DELETE CASCADE,
  
  comp_type TEXT NOT NULL, -- 'pre_accident' | 'post_accident'
  source TEXT NOT NULL, -- 'auto_search' | 'manual_entry'
  
  -- Vehicle Details
  vin TEXT NOT NULL,
  year INTEGER NOT NULL,
  make TEXT NOT NULL,
  model TEXT NOT NULL,
  trim TEXT NOT NULL,
  mileage INTEGER NOT NULL,
  accident_history TEXT NOT NULL, -- 'no_accidents' | 'accident_reported'
  
  -- Listing Info
  listing_url TEXT,
  listing_price NUMERIC(10, 2) NOT NULL,
  dealer_name TEXT,
  dealer_phone TEXT,
  location_city TEXT,
  location_state TEXT,
  distance_miles INTEGER,
  
  -- Adjustments (JSONB for complex structure)
```

#### 4.3.1 Comparable Vehicle Adjustment Calculator
```typescript
// Function: calculateCompAdjustments()
// Called for each comparable vehicle to determine adjusted value

interface CompAdjustment {
  mileage_adjustment: number;
  equipment_adjustments: number;
  trim_adjustment: number;
  year_adjustment: number;
  condition_adjustment: number;
  total_adjustments: number;
  adjusted_value: number;
}

function calculateCompAdjustments(
  subjectVehicle: SubjectVehicle,
  comp: ComparableVehicle
): CompAdjustment {
  
  const adjustments: CompAdjustment = {
    mileage_adjustment: 0,
    equipment_adjustments: 0,
    trim_adjustment: 0,
    year_adjustment: 0,
    condition_adjustment: 0,
    total_adjustments: 0,
    adjusted_value: comp.listing_price
  };
  
  // 1. MILEAGE ADJUSTMENT
  // Industry standard: $0.12 per mile difference
  // Positive if comp has MORE miles (increases comp value to match subject)
  const mileageDiff = comp.mileage - subjectVehicle.mileage_at_accident;
  adjustments.mileage_adjustment = mileageDiff * -0.12; // Negative diff = positive adjustment
  
  // 2. EQUIPMENT ADJUSTMENTS
  // Compare subject's optional equipment vs comp
  let equipmentTotal = 0;
  
  // Subject has equipment that comp doesn't → SUBTRACT from comp
  subjectVehicle.optional_equipment.forEach(item => {
    if (!comp.equipment_list.includes(item.name)) {
      equipmentTotal -= item.factory_price * 0.80; // 80% of MSRP
    }
  });
  
  // Comp has equipment that subject doesn't → ADD to comp
  comp.additional_equipment.forEach(item => {
    equipmentTotal += item.adjustment_value;
  });
  
  adjustments.equipment_adjustments = equipmentTotal;
  
  // 3. TRIM ADJUSTMENT (only if not exact match)
  if (subjectVehicle.trim !== comp.trim) {
    // Use NADA/KBB data for trim MSRP differences
    // For now, use estimated adjustment based on trim level
    adjustments.trim_adjustment = calculateTrimDifference(
      subjectVehicle.make,
      subjectVehicle.model,
      subjectVehicle.trim,
      comp.trim
    );
  }
  
  // 4. YEAR ADJUSTMENT (if years differ)
  if (subjectVehicle.year !== comp.year) {
    const yearDiff = comp.year - subjectVehicle.year;
    // ~7% depreciation per year for vehicles under 5 years old
    adjustments.year_adjustment = comp.listing_price * (yearDiff * -0.07);
  }
  
  // 5. CONDITION ADJUSTMENT
  // Standardize all comps to "Good" condition for comparison
  // Then adjust subject vehicle separately
  const conditionMultipliers = {
    'excellent': -0.05,  // Reduce comp value by 5% to match "good"
    'good': 0,           // No adjustment needed
    'average': 0.05,     // Increase comp value by 5% to match "good"
    'below_average': 0.10,
    'rough': 0.15
  };
  
  if (comp.condition && comp.condition !== 'good') {
    adjustments.condition_adjustment = 
      comp.listing_price * conditionMultipliers[comp.condition];
  }
  
  // TOTAL ADJUSTMENTS
  adjustments.total_adjustments = 
    adjustments.mileage_adjustment +
    adjustments.equipment_adjustments +
    adjustments.trim_adjustment +
    adjustments.year_adjustment +
    adjustments.condition_adjustment;
  
  // FINAL ADJUSTED VALUE
  adjustments.adjusted_value = comp.listing_price + adjustments.total_adjustments;
  
  return adjustments;
}
```

#### 4.3.2 Diminished Value Calculator
```typescript
// Function: calculateDiminishedValue()
// Main calculation engine

interface DVCalculationResult {
  pre_accident_fmv: number;
  post_repair_acv: number;
  diminished_value: number;
  dv_percentage_of_pre_value: number;
  dv_percentage_of_repair_cost: number;
  calculation_method: string;
  confidence_range: {
    low: number;
    high: number;
  };
}

function calculateDiminishedValue(
  preAccidentComps: ComparableVehicle[],
  postAccidentComps: ComparableVehicle[],
  repairCost: number
): DVCalculationResult {
  
  // Filter to only included comps
  const includedPreComps = preAccidentComps.filter(c => c.included_in_calculation);
  const includedPostComps = postAccidentComps.filter(c => c.included_in_calculation);
  
  // Calculate adjusted values for all comps
  const preAdjustedValues = includedPreComps.map(c => c.adjusted_value);
  const postAdjustedValues = includedPostComps.map(c => c.adjusted_value);
  
  // PRE-ACCIDENT FMV: Use MEDIAN (more resistant to outliers)
  const preAccidentFMV = calculateMedian(preAdjustedValues);
  
  // POST-REPAIR ACV: Use MEDIAN
  const postRepairACV = calculateMedian(postAdjustedValues);
  
  // DIMINISHED VALUE
  const diminishedValue = preAccidentFMV - postRepairACV;
  
  // Calculate as percentages
  const dvPercentageOfPreValue = (diminishedValue / preAccidentFMV) * 100;
  const dvPercentageOfRepairCost = (diminishedValue / repairCost) * 100;
  
  // Confidence range (10th to 90th percentile)
  const confidenceLow = calculatePercentile(preAdjustedValues, 10) - 
                        calculatePercentile(postAdjustedValues, 90);
  const confidenceHigh = calculatePercentile(preAdjustedValues, 90) - 
                         calculatePercentile(postAdjustedValues, 10);
  
  return {
    pre_accident_fmv: preAccidentFMV,
    post_repair_acv: postRepairACV,
    diminished_value: diminishedValue,
    dv_percentage_of_pre_value: dvPercentageOfPreValue,
    dv_percentage_of_repair_cost: dvPercentageOfRepairCost,
    calculation_method: "Comparable Sales Method (Median)",
    confidence_range: {
      low: Math.max(0, confidenceLow),
      high: confidenceHigh
    }
  };
}

// Helper: Calculate median
function calculateMedian(values: number[]): number {
  const sorted = [...values].sort((a, b) => a - b);
  const mid = Math.floor(sorted.length / 2);
  return sorted.length % 2 === 0
    ? (sorted[mid - 1] + sorted[mid]) / 2
    : sorted[mid];
}

// Helper: Calculate percentile
function calculatePercentile(values: number[], percentile: number): number {
  const sorted = [...values].sort((a, b) => a - b);
  const index = (percentile / 100) * (sorted.length - 1);
  const lower = Math.floor(index);
  const upper = Math.ceil(index);
  const weight = index - lower;
  return sorted[lower] * (1 - weight) + sorted[upper] * weight;
}
```

```typescript
// Function: classifyDamageSeverity()
// Auto-determines severity level 1-5 based on repair data

interface SeverityClassification {
  severity_level: 1 | 2 | 3 | 4 | 5;
  severity_text: string;
  justification: string;
  post_repair_naaa_grade: string;

function classifyDamageSeverity(
  accidentDetails: AccidentDetails,
  subjectVehicle: SubjectVehicle
): SeverityClassification {
  
  let severityLevel: 1 | 2 | 3 | 4 | 5 = 1;
  let justification = "";
  
  const {
    total_labor_hours,
    frame_pulling_required,
    frame_labor_hours,
    structural_damage,
    airbag_deployment,
    panels_replaced
  } = accidentDetails;
  
  // DECISION TREE LOGIC
  
  // LEVEL 5: SEVERE (Most restrictive conditions first)
  if (
    total_labor_hours > 60 ||
    (airbag_deployment && structural_damage && frame_labor_hours > 5) ||
    (frame_labor_hours > 10)
  ) {
    severityLevel = 5;
    justification = `This repair is classified as SEVERE (Level 5) due to: ${
      total_labor_hours > 60 ? `excessive total labor hours (${total_labor_hours}hrs)` : ''
    }${
      frame_labor_hours > 10 ? `, extensive frame work (${frame_labor_hours}hrs on frame machine)` : ''
    }${
      airbag_deployment && structural_damage ? ', airbag deployment combined with structural damage' : ''
    }. This level of damage indicates catastrophic impact with multiple critical systems compromised.`;
  }
  
  // LEVEL 4: MAJOR
  else if (
    frame_pulling_required ||
    frame_labor_hours > 0 ||
    (structural_damage && total_labor_hours > 35) ||
    (airbag_deployment && total_labor_hours > 30)
  ) {
    severityLevel = 4;
    justification = `This repair is classified as MAJOR (Level 4) due to: ${
      frame_pulling_required ? 'documented frame pulling operations' : ''
    }${
      frame_labor_hours > 0 ? `, frame machine work (${frame_labor_hours}hrs)` : ''
    }${
      structural_damage ? ', replacement of primary structural components' : ''
    }${
      airbag_deployment ? ', airbag deployment indicating high-impact forces' : ''
    }. The unibody structure was deformed and required specialized equipment to restore factory dimensions, permanently altering the vehicle's original structural integrity.`;
  }
  
  // LEVEL 3: MEDIUM
  else if (
    total_labor_hours >= 20 && total_labor_hours <= 35 ||
    (panels_replaced.some(p => p.panel_type === 'structural') && total_labor_hours > 15) ||
    airbag_deployment
  ) {
    severityLevel = 3;
    justification = `This repair is classified as MEDIUM (Level 3) due to: moderate total labor hours (${total_labor_hours}hrs)${
      panels_replaced.filter(p => p.panel_type === 'structural').length > 0 
        ? ', replacement of structural panels' 
        : ''
    }${
      airbag_deployment ? ', airbag deployment' : ''
    }. While no frame pulling was required, the extent of repairs and panel replacements indicates significant impact forces that may have affected adjacent components.`;
  }
  
  // LEVEL 2: MODERATE
  else if (
    total_labor_hours >= 10 && total_labor_hours < 20 &&
    !structural_damage &&
    !frame_pulling_required
  ) {
    severityLevel = 2;
    justification = `This repair is classified as MODERATE (Level 2) due to: limited labor hours (${total_labor_hours}hrs) with no structural or frame damage. Repairs were limited to bolt-on panels and cosmetic components. However, the vehicle's history now includes documented collision repairs, which will affect market perception.`;
  }
  
  // LEVEL 1: MINOR
  else {
    severityLevel = 1;
    justification = `This repair is classified as MINOR (Level 1) due to: minimal labor hours (${total_labor_hours}hrs) involving only cosmetic damage with no structural components affected. Despite professional repairs, the permanent collision history record will still impact resale value.`;
  }
  
  // DETERMINE POST-REPAIR NAAA GRADE
  let postRepairNaaaGrade = "";
  let naaaExplanation = "";
  
  const preGrade = subjectVehicle.pre_accident_condition.overall_grade;
  
  if (severityLevel === 5) {
    postRepairNaaaGrade = "1 - Rough";
    naaaExplanation = "Severe structural damage and extensive repairs prevent this vehicle from achieving any grade higher than Rough (1), regardless of repair quality. The permanent alteration of factory-original structural integrity and the extent of component replacement fundamentally change the vehicle's condition classification.";
  } else if (severityLevel === 4) {
    postRepairNaaaGrade = "2 - Below Average";
    naaaExplanation = "According to NAAA standards, any vehicle with documented structural repairs, frame pulling, or primary component replacement cannot be rated higher than Below Average (2), even when repairs are completed by certified technicians using OEM parts. The factory-original condition cannot be fully restored once structural work is performed.";
  } else if (severityLevel === 3) {
    postRepairNaaaGrade = "3 - Average";
    naaaExplanation = "With moderate damage and multiple panel replacements, this vehicle can achieve a maximum grade of Average (3) post-repair. While no frame work was required, the extent of repairs prevents classification as Good or Excellent.";
  } else if (severityLevel === 2) {
    postRepairNaaaGrade = "4 - Good";
    naaaExplanation = "Moderate cosmetic damage with professional repairs allows this vehicle to potentially achieve a Good (4) grade, provided all work was completed to industry standards and no repair-related defects are present.";
  } else {
    postRepairNaaaGrade = preGrade === 'excellent' ? "4 - Good" : "3 - Average";
    naaaExplanation = "Minor cosmetic damage with proper repairs may allow this vehicle to achieve Good (4) or Average (3) condition, though it can never return to Excellent grade once collision history is documented.";
  }
  
  const severityTextMap = {
    1: "Minor",
    2: "Moderate", 
    3: "Medium",
    4: "Major",
    5: "Severe"
  };
  
  return {
    severity_level: severityLevel,
    severity_text: severityTextMap[severityLevel],
    justification: justification,
    post_repair_naaa_grade: postRepairNaaaGrade,
    naaa_grade_explanation: naaaExplanation
  };
}
```

```typescript
      const listings = parseListings(html, source);
      
      // Filter listings by criteria
      const filtered = listings.filter(listing => {
        // Mileage within tolerance
        const mileageMatch = Math.abs(listing.mileage - params.mileage) <= params.mileage_tolerance;
        
        // Accident history match
        const accidentMatch = params.accident_history_required 
          ? listing.accident_history === 'accident_reported'
          : listing.accident_history === 'no_accidents';
        
        // Year/make/model/trim match (exact or very close)
        const vehicleMatch = (
          listing.year === params.year &&
          listing.make.toLowerCase() === params.make.toLowerCase() &&
          listing.model.toLowerCase() === params.model.toLowerCase() &&
          (listing.trim.toLowerCase().includes(params.trim.toLowerCase()) ||
           params.trim.toLowerCase().includes(listing.trim.toLowerCase()))
        );
        
        return mileageMatch && accidentMatch && vehicleMatch;
      });
      
      results.push(...filtered);
      
      // Add delay between requests to avoid rate limiting
      await sleep(2000);
      
    } catch (error) {
      console.error(`Error scraping ${source}:`, error);
      // Continue to next source
    }
  }
  
  // If insufficient results, expand search radius or mileage tolerance
  if (results.length < params.min_results) {
    console.warn(`Only found ${results.length} comps, expanding search...`);
    // Recursive call with relaxed params
    return searchComparableVehicles({
      ...params,
      mileage_tolerance: params.mileage_tolerance * 1.5,
      search_radius_miles: params.search_radius_miles * 1.5
    });
  }
  
  // Return top results sorted by relevance
  return results
    .sort((a, b) => {
      // Prioritize closer mileage match
      const aMileageDiff = Math.abs(a.mileage - params.mileage);
      const bMileageDiff = Math.abs(b.mileage - params.mileage);
      return aMileageDiff - bMileageDiff;
    })
    .slice(0, params.max_results);
}

function buildSearchUrl(source: string, params: CompSearchParams): string {
  const baseUrls = {

  };

  let url = baseUrls[source] || '';
```

```typescript
          of the liability or $5,000 (whichever is greater), plus reasonable attorney fees. 
          <em>State Farm Mut. Auto Ins. Co. v. Mabry</em>, 274 Ga. 498 (2001).
        </p>
        <p style="margin: 10px 0;">
          <strong>Third-Party Claims (O.C.G.A. § 33-4-7):</strong> Insurance companies issuing motor vehicle 
          liability policies have an affirmative duty to adjust property losses fairly and promptly, including 
          diminished value. Bad faith handling may result in penalties of up to 50% of liability or $5,000 
          (whichever is greater), plus attorney fees.
        </p>
        <p style="margin: 10px 0;">
          <strong>Georgia Insurance Commissioner Directive (December 2008):</strong> The Georgia Department 
          of Insurance has never endorsed any specific formula for calculating diminished value, including 
          the "17c formula." Insurers must consider all relevant information provided by the insured when 
          evaluating diminished value claims.
        </p>
      </div>
    `,
    'NC': `
      <div class="info-box">
        <strong>North Carolina Diminished Value Law:</strong>
        <p style="margin: 10px 0;">
          <strong>Appraisal Process (N.C. Gen. Stat. § 20-279.21(d)(1)):</strong> When the insured and 
          insurer disagree on diminished value by more than $2,000 or 25% of the vehicle's fair market 
          value, each party shall select an independent appraiser. If they still cannot agree, a third 
          appraiser renders a final and binding verdict.
        </p>
        <p style="margin: 10px 0;">
          North Carolina courts recognize diminished value claims in both first-party and third-party 
          contexts, with the burden of proof resting on the claimant to demonstrate actual loss in market value.
        </p>
      </div>
    `,
    'default': `
      <div class="info-box">
        <strong>General Tort Law Principles:</strong>
        <p style="margin: 10px 0;">
          Under general tort law, a claimant is entitled to be made whole following damage to their property. 
          This includes compensation for the difference between the property's value before and after the loss, 
          even when repairs are completed. Many states recognize diminished value claims in third-party contexts, 
          with some variation in first-party coverage requirements.
        </p>
        <p style="margin: 10px 0;">

      </div>
    `
  };
  
  return citations[state] || citations['default'];
}

// Helper: Format date
function formatDate(date: Date | string): string {
  const d = typeof date === 'string' ? new Date(date) : date;
  return d.toLocaleDateString('en-US', { 
    year: 'numeric', 
    month: 'long', 
    day: 'numeric' 
  });
}
```
